# NestJS Scraper Test

Proyecto de prueba para hacer scraping usando:

- NestJS
- Node.js (LTS)
- PNPM
- Docker

> Proyecto solo para pruebas, no pensado para producción.

## Levantar el proyecto

```bash
docker compose up --build
Detener el proyecto
docker compose down
```

## Acceso

http://localhost:3000

## Descripción

Este proyecto se usa únicamente para practicar scraping dentro de un backend con NestJS usando Docker.
